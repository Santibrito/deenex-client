# Magic UI Skill

When generating UI:

- prefer modern SaaS layouts
- clean spacing
- large rounded corners
- smooth hover animations
- soft shadows
- minimal borders
- modern typography

Component style inspiration:

- Stripe
- Linear
- Vercel
- Raycast
- Notion

UI guidelines:

- use TailwindCSS utilities
- use flex and grid layouts
- prioritize whitespace
- minimalistic icons
- premium SaaS feels
